# Infrared controlled robot

A moving robot with 2 DC motors controlled by an infrared remote.

## Installation

Use the file and the library located in the folder *ir_commanded_robot*.

## Pins

Pins used for the motors.

### Motor 1 (left)

Constant         | Pin number
-----------------|-----------
PIN_ACTIVATE_1_2 | 9
PIN_INPUT_1      | 10
PIN_INPUT_2      | 11

### Motor 2 (right)

Constant         | Pin number
-----------------|-----------
PIN_ACTIVATE_3_4 | 3
PIN_INPUT_3      | 4
PIN_INPUT_4      | 5
